Payments API

0. Логін
POST /auth/token
1. Методи оплати
GET /payments/methods — публічно
2. Створити метод оплати
POST /payments/methods — тільки ADMIN
3. Оновити метод оплати
PATCH /payments/methods/{method_id} — тільки ADMIN
4. Видалити метод оплати
DELETE /payments/methods/{method_id} — тільки ADMIN
5. Всі платежі
GET /payments/ — тільки ADMIN
6. Платіж по ID
GET /payments/{payment_id}
7. Платежі по оренді
GET /payments/rental/{rental_id}
8. Створити платіж
POST /payments/ — transaction_id генерується автоматично
9. Змінити статус платежу
PATCH /payments/{payment_id}/status — тільки ADMIN, при COMPLETED встановлюється paid_at
