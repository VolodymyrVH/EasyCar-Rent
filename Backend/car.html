Cars API

0. Логін
POST /auth/token
1. Всі авто
GET /cars/?skip=0&limit=100
2. Авто по ID
GET /cars/{car_id}
3. Авто по бренду
GET /cars/brand/{brand_id}
4. Авто по статусу
GET /cars/status/{status}
5. Авто по типу
GET /cars/type/{car_type_id}
6. Створити авто
POST /cars/ — тільки ADMIN
7. Оновити авто
PATCH /cars/{car_id} — тільки ADMIN
8. Змінити статус авто
PATCH /cars/{car_id}/status — тільки ADMIN
9. Додати зображення
POST /cars-details/cars/{car_id}/images — тільки ADMIN
10. Зображення авто
GET /cars-details/cars/{car_id}/images
11. Видалити авто
DELETE /cars/{car_id} — тільки ADMIN
